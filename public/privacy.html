Privacy Policy – QUE VAUT TON NOM ?

QUE VAUT TON NOM ? is an interactive web application designed for TikTok Live streamers.

This application uses the official TikTok Live Interaction API to receive real-time live events such as messages, gifts, likes, and viewer joins during live sessions.

Data Collection

The application does not collect, store, or share any personal user data outside of the real-time TikTok Live session.

Data Usage

Live event data received from TikTok is used only to trigger on-screen visual effects, scores, and animations during the live stream.

Data Storage

No TikTok user data is stored on any server. Temporary game data may be stored locally in the browser of the streamer for gameplay continuity.

Data Security

The application follows TikTok Developer Policies and processes only officially provided event data.

User Rights

Since no personal data is stored, users are not subject to data retention or tracking.
